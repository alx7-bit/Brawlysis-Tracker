Live API Syncing Configuration

The Supercell API cannot be called from the browser alone (no CORS). This app uses a tiny same-site proxy (/api/official/…) when you deploy to Vercel or Netlify. That proxy forwards through RoyaleAPI’s Brawl Stars proxy so your key is not tied to Vercel’s random server IPs. GitHub Pages has no proxy — deploy the same repo to Vercel/Netlify, or paste that HTTPS URL below. Your token still comes from developer.brawlstars.com (follow their IP rules for your setup).

Leave empty when you open the app on Vercel/Netlify. If you use github.io, paste the HTTPS root of a Vercel (or Netlify) deployment of this same project.
